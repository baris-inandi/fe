# fe

Install using:
```
paru -S fe
yay -S fe
```

```

NAME:
   fe - AUR helper with a familiar subcommand system

USAGE:
   fe [global options] command [command options] [arguments...]

COMMANDS:
   install, in    Syncs packages
   remove, rm     Removes packages
   search, se     Searches for packages using given keyword
   update, up     Updates all packages
   clean, cl      Removes unnecessary packages
   query, qr      Queries installed packages
   fuzzy, fz      Fuzzy-search for packages using paruz
   configure, cf  Overwrite /etc/paru.conf with recommended configuration
   help, h        Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --default, -d           Don't implicitly pass flags to paru (default: false)
   --help, -h              show help (default: false)
   --noparu, -n            Use pacman instead of paru (default: false)
   --paru value, -p value  Pass arbitrary options to paru
   
```
